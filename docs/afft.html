Tentang AFFT

AFFT adalah kerangka kerja untuk alat forensik yang membantu Anda melakukan berbagai tugas seperti scanning website (subdomain finder, direktori) dan alat-alat forensik lainnya.

Fitur Utama

  • Subdomain Finder
  • Direktori Scanner
  • Alat Forensik Lainnya

Cara Penggunaan

  1. Unduh file AFFT dari link di bawah.
  2. Ekstrak file ZIP.
  3. Jalankan perintah sesuai dengan kebutuhan alat Anda.
